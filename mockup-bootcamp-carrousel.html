Partie A · Audience manquante

Créer l'audience courriels (la seule manquante)

A1
Custom Audience — Liste courriels courtiers
  1. Ouvre Meta Business Suite → business.facebook.com
  2. Menu gauche → Tous les outils → Audiences
  3. Bouton bleu Créer une audience → Audience personnalisée
  4. Sélectionne Liste de clients comme source
  5. Prépare ton CSV avec colonnes : email, prénom, nom, téléphone (plus tu donnes de champs, meilleur le match)
  6. Glisse-dépose ton CSV, accepte les conditions, clique Importer et créer
  7. Nomme-la : BOOTCAMP_Courtiers_Courriels
  8. Attends 30-60 min que Meta fasse le match (taux attendu 70-90 %)
💡 Astuce : ajoute une colonne code postal et ville pour booster le taux de match.
A2
Note sur le groupe Facebook interne
  1. ⚠️ Meta ne permet PAS de cibler les membres d'un groupe directement
  2. Bonne nouvelle : tes audiences Interaction page FB 365j + Interaction page IG 365j couvrent déjà la majorité des courtiers actifs sur tes pages
  3. Action gratuite : publie le carrousel en post organique dans le groupe FB le même jour que le lancement de Phase 1 — double exposition sans coût
Partie B · Phase 1 — Notoriété (juin)

Créer la campagne Notoriété

B1
Niveau Campagne
  1. Gestionnaire de publicités → + Créer
  2. Objectif : Notoriété (Awareness)
  3. Nom campagne : BOOTCAMP_2026_P1_AWARENESS
  4. Catégorie de pub spéciale : Aucune
  5. Budget : CBO désactivé (budget au niveau adset)
  6. A/B test : non
B2
Niveau Ensemble de publicités (Adset)
  1. Optimisation : Couverture (reach)
  2. Budget : 10 $/jour × ~30 jours = 300 $
  3. Dates : 1 juin → 30 juin 2026
  4. Audience (combinées avec OR) :
    • BOOTCAMP_Courtiers_Courriels (à créer A1)
    • Interaction - page fb-365 jours ✅
    • Interaction - page IG - 365 jours ✅
  5. Lieux : Québec (sécurité, l'audience custom restreint déjà)
  6. Âge : 25-65+ · Langue : français
  7. Plafond de fréquence : 3 impressions / 7 jours
  8. Placements : Manuels → Facebook Feed + Instagram Feed + Stories seulement
B3
Niveau Publicité (créatif)
  1. Format : Image unique
  2. Visuel : 1.png (cover BOOT CAMP du mockup)
  3. Texte principal : « Notre Boot Camp 2026 aura lieu les 14 et 15 octobre. 2 jours, 8 conférenciers de haut niveau. Réserve la date. »
  4. Titre : « Boot Camp RE/MAX Crystal d'Ici »
  5. Description : « 14 & 15 octobre 2026 »
  6. CTA : En savoir plus
  7. URL destination : lien Eventbrite du Boot Camp
  8. Suivi : ajoute UTM au lien Eventbrite ?aff=meta_awareness (le param aff est suivi nativement par Eventbrite dans son dashboard)
Partie C · Phase 2 — Trafic (juillet)

Créer la campagne Trafic (carrousel 8 conférenciers)

C1
Audience retargeting Phase 1
  1. Avant de créer la campagne C2, crée une nouvelle Custom Audience
  2. Audiences → Créer → Engagement → Publicité Instagram/Facebook
  3. « Personnes ayant vu ou cliqué sur ta publicité » → sélectionne la campagne BOOTCAMP_2026_P1_AWARENESS
  4. Durée : 30 jours
  5. Nomme : BOOTCAMP_Retarget_P1_Engaged
C2
Campagne Trafic
  1. + Créer → Objectif : Trafic
  2. Nom : BOOTCAMP_2026_P2_TRAFIC
  3. Adset → Destination : Site web (lien Eventbrite)
  4. Optimisation : Clics sur le lien
  5. Budget : 10 $/jour · Dates : 1 → 31 juillet
  6. Audience principale (combinées avec OR) :
    • BOOTCAMP_Retarget_P1_Engaged (créée en C1)
    • visiteurs page web-180 jours ✅ (déjà créée)
  7. Audience exclusion : aucune pour l'instant
  8. Placements : Manuels → Feeds + Stories
C3
